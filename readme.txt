ჩვენი აპლიკაციის,სახელად "PInfo"-ს,მთავარი გამოყენება მომხმარებლის ინფორმაციის შენახვაა.
პირველ რიგში ხდება მომხმარებლის რეგისტრაცია Email-ით და პაროლით.
წარმატებული რეგისტრაციის შემდგომ მომხმარებელი ავტომატურად აღმოჩნდება გვერდზე სადაც მას დახვდება ინფორმაციის შესაყვანი ველები : Name/Surname,Age,Hobbies,Picture Link,Status,Workplace.
ველების შეყვანის შემდგომ ქვევით მოთავსებული Save ღილაკის მეშვეობით ინფორმაცია ინახება Realtime Database-ზე.
Navigation Component-ის დახმარებით მომხმარებელი გადავა შემდეგ გვერდზე სადაც მას დახვდება შენახული ინფორმაცია,რომლის არ არსებობის შემთხვევაშიც გამოისახება Default მნიშვნელობები.
ამავე გვერდზე არის Sign Out და User List ღილაკები.
User List-ზე დაჭერის შემთხვევაში მომხმარებელი გადადის Recycler View-ზე სადაც გამოსახული იქნება ყველა დარეგისტრირებული პირი და მათ მიერ შეყვანილი ინფორმაცია.
პაროლის დავიწყების შემთხვევაში აპლიკაციას Login Page-ზე ასევე გააჩნია Forgot Password ღილაკი,რომელზე დაჭერის შემთხვევაშიც მომხმარებელს შეჰყავს Email,სადაც მიუვა აღდგენის ლინკი.
